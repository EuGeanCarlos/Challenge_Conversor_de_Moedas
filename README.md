# Challenge_Conversor_de_Moedas
Projeto desenvolvido como parte do Praticando Java: Challenge Conversor de Moedas, da Alura, em parceria com o programa Oracle Next Education (ONE).  O desafio tem como objetivo consolidar conhecimentos em Java, Programação Orientada a Objetos, consumo de API externa, manipulação de JSON e boas práticas de versionamento com Git/GitHub.
